# Website Kuis Online Sederhana

## Deskripsi Project
Website kuis online sederhana berbasis HTML, CSS, dan JavaScript.
Project ini menampilkan soal pilihan ganda dan menghitung skor secara otomatis tanpa menggunakan backend atau database.

## Alur Logika / Algoritma
1. User membuka halaman website kuis
2. Website menampilkan 5 soal pilihan ganda
3. User memilih satu jawaban pada setiap soal
4. Ketika tombol submit diklik:
   - JavaScript mengecek jawaban user
   - Setiap jawaban benar menambah skor 20 poin
5. Total skor dihitung

9. Tombol ulangi digunakan untuk mereset jawaban dan hasil kuis
